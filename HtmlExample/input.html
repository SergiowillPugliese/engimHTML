Esempi di Tag <input>

1. Input di testo:

Un campo di testo standard per l'inserimento di testo.

2. Input password:

Un campo di testo mascherato utilizzato per inserire password.

3. Casella di controllo (checkbox):

Una casella di controllo per selezionare o deselezionare un'opzione.

4. Pulsante di opzione (radio button):

Pulsanti di opzione per selezionare una sola opzione da un gruppo.

5. Campo di selezione (select):

Un menu a discesa per selezionare un'opzione da un elenco.
